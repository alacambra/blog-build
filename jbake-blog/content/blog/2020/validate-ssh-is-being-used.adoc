== Validate signature key
.Show fingerprint
[source, shell]
----
> ssh-keygen -lf /path/to/used/key.pub

.Show fingerprint with md5
[source, shell]
----
> ssh-add -l -E md5
4096 MD5:23:g5:2f:eb:sd:76:43:qw:35:ce:4w:05:23:71:8d:d4 /path/to/used/key (RSA)
----

== Load key
[source, shell]
----
> eval $(ssh-agent -s)
> ssh-add ~/.ssh/${PRIVATE_KEY}
----

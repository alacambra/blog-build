Having:

[source, java]
--
public void setX to become a public SomeClass setX(â€¦)
--

and using the regex:
[source, java]
--
public void set([\w]+.+\{(\n\s+).+)
--

with replace:
[source, java]
--
public SomeClass set$1$2return this;
--


It transforms
[source, java]
--
public void setX(int x){
    this.x;
}
--

to
[source, java]
--
public MyClass setX(int x){
    this.x;
    return this;
}
--
